Referință API

Toate endpoint-urile returnează JSON. Autentificare: nu este necesară (date publice).

Base URL: https://cdep.ro/api/v1 propus
GET /deputati Listă completă deputați ▾
Returnează lista tuturor deputaților din legislatura curentă sau specificată, cu date despre prezență și activitate.
Parametri
ParametruTipDescriere
legislaturaintegeropționalAnul de start al legislaturii. Ex: 2024
partidstringopționalFiltrare după partid: PSD, PNL, USR, AUR...
judetstringopționalFiltrare după județul de circumscripție
pageintegeropționalPagina rezultatelor. Default: 1
per_pageintegeropționalRezultate pe pagină. Default: 20, max: 100
Testează
GET cdep.ro/api/v1/deputati?legislatura=2024&partid=USR
